# PDF RAG System

A Retrieval-Augmented Generation (RAG) pipeline using:

- LangChain
- ChromaDB
- Sentence Transformers
- OpenRouter LLMs

## Features

- PDF ingestion
- Text chunking
- Embedding generation
- Vector search
- Semantic retrieval

## Tech Stack

- Python
- ChromaDB
- SentenceTransformers
- LangChain
